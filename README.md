### How to install

1. **_Repository Clone_**

   ```bash
   git clone https://github.com/Informatics-ITS/TA.git
   ```

2. **_Dependency Installation_**

   ```bash
   cd [folder-proyek]
   npm install
   ```

3. **Setup**

- Copy or rename the .env.example file to .env

  ```bash
  PORT=
  JWT_SECRET=
  DB_HOST=
  DB_PORT=
  DB_USER=
  DB_PASSWORD=
  DB_NAME=
  SEED_SUPERADMIN=
  NODE_ENV=
  ```

4. **_Start_**

   ```bash
   cd [folder-proyek]
   npm run dev
   ```
